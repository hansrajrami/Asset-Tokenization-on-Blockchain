# KPRIET NFT Demo 

Compile
```
npx hardhat compile
```

Test
```
npx hardhat test
```

Deploy
```
npx hardhat run scripts/deployProxy.js --network mumbai
```

[Polygon Mumbai Testnet Block Explorer link](https://mumbai.polygonscan.com/)

Verify
```
npx hardhat verify --network mumbai <contract address>
```
